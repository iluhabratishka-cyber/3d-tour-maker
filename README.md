# 3D Тур из Фото

Простой сервис для создания интерактивных 3D туров из обычных фотографий дома.

## 🎯 Основная идея

Берём обычные фотографии комнат и превращаем их в виртуальный тур, который можно просматривать онлайн.

## ✨ Функции

- 📸 Загрузка фотографий комнат
- 🎮 Интерактивный просм
- 🔄 Переключение между фото
- 📱 Адаптивный дизайн

## 🚀 Как начать

1. Клонируйте репозитори
2. Откройте `index.html` в браузере
3. Загрузите фото комнат
4. Создавайте виртуальные туры!

## 📁 Структура проекта

- `index.html` - Главная страница
- `style.css` - Стили оф
- `script.js` - Логика 3D тура
